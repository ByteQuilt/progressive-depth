# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :x:                |
| < 1.1   | :white_check_mark: |

## Reporting a Vulnerability

Feel free to report any vulnerabilities to https://github.com/ByteQuilt/progressive-depth/issues or hello@bytequilt.com.
